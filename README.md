# React Native Boilerplate

A minimal React Native boilerplate using Expo and TypeScript.

## Features

- Expo Router for file-based navigation
- Dark/Light theme switching
- Tab navigation with Home and Profile screens
- Custom fonts (JetBrains Mono)
- TypeScript support
- Safe area handling

## Installation

1. Clone the repository
2. Install dependencies: `yarn install` or `npm install`
3. Start the app: `yarn start` or `npm start`

## Usage

- Switch themes in the Profile tab
- Customize themes in `constants/theme.ts`
- Add new screens in `app/(tabs)/`

## Scripts

- `start`: Start Expo development server
- `android`: Start on Android emulator/device
- `ios`: Start on iOS simulator/device
- `web`: Start on web browser
